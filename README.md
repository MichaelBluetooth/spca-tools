# SPCA Tools

A set of tools to quickly validate reports generated by PetPoint.

# Setup

```
git clone https://github.com/MichaelBluetooth/spca-tools.git
cd spca-tools
npm install
npm run build
```

# Run the program

```
node .\dist\main.js ./data/MyAnimalIntakeExtendedReport.csv
```
